Files to create to build the app:
- build-local.properties
	- for example like this:
	- jar.javafx=${env.JAVA_HOME}/jre/lib/jfxrt.jar
	- jar.ant-javafx=${env.JAVA_HOME}/lib/ant-javafx.jar
- nbproject/build.properties
	- for example like this:
	- jar.javafx=D:/Programy/Java/jdk1.8.0/jre/lib/jfxrt.jar
	- jar.jfxtras=c:/Users/hujeceka/Documents/NetBeansProjects/AutoPolAn2/lib/jfxtras-labs-8.0-r1-20131027.115451-7.jar
	- jar.controlsfx=c:/Users/hujeceka/Documents/NetBeansProjects/AutoPolAn/Client/lib/controlsfx-8.0.2.jar
